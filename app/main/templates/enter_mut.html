Mutation Entry Guidelines

  • Substitutions: Enter substitutions in the format A512D, where the original amino acid appears first, followed by the position number and the substituted amino acid.
  • Deletions: Enter deletions using the format XG510Q518, where X marks a deletion, followed by each AA deleted and position of deletion(s).
  • Insertions: Enter insertions using the format ZD516Q517, where Z marks an insertion, followed by each AA inserted and position of insertion(s).
  • Base pair changes: Record base pair substitutions using two-letter notation with the original base first and the mutated base second (coding strand changes only).
    Example: AC (A → C).
  • Multiple base pair changes: Separate multiple changes with commas and no spaces.
    Example: CT,AG,TA
  • Unknown base pair changes: If the base pair change is unknown, enter unk.
  • Source citation: If the mutation was experimentally determined, list the contributor(s) in the format:
    Name(s) or Group Number, Course Number Term+Year
    Example: Group 1, BB1801 C25

    If found in published research article, enter doi link.
